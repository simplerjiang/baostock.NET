# reference/baostock-python

**Read-only snapshot** of the official Python `baostock` package
(installed from `pip install baostock` on conda env at the time of import).

Purpose: protocol reverse-engineering reference for `Baostock.NET`.
Do **not** modify these files. Do **not** ship them. They are excluded from
the .NET build and exist only as a vendored doc artifact for developers.

Original source: `pip install baostock` — see https://baostock.com/
Upstream license: see baostock's own terms; this snapshot is for
interoperability research, not redistribution.
